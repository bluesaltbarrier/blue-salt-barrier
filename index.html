Notice — 2026-04-18

The Blue Salt Barrier v1 has been withdrawn.

What happened

A post-submission code audit revealed that the MPAS atmospheric model driver never passed the giant-CCN (GCCN) tracer array to the Thompson microphysics subroutine. The GCCN lifecycle code compiled and was present in the source tree, but was never executed at run time.

Every result reported in the v1 paper and website — transport changes, Antarctic cooling, equatorial rainfall shifts — reflects a Twomey aerosol-cloud indirect effect driven by a 5–10× boost to the standard water-friendly aerosol field over rainforest cells, not the giant-CCN warm-rain seeding mechanism the text described. The numerical outputs are real MPAS simulations, but the mechanism attribution is wrong.

What is being done

  • The EarthArXiv v1 preprint submission has been withdrawn before DOI assignment.
  • The coupling has been patched: ngccn and ngccn2d are now plumbed through mpas_atmphys_interface.F, passed to mp_gt_driver, and emitted as a per-timestep surface source in the Thompson microphysics module.
  • Paired January and July 120 km runs with the GCCN pathway actually active are running now.
  • A revised paper and site will be posted once those results are analyzed.

What readers should do

Please do not cite v1 numbers. The mechanism described in that version is not the mechanism the model executed. If you have already cited or built on v1 results, please wait for the corrected version and contact the author.

Archive

The v1 text and figures, as submitted to EarthArXiv, are preserved at git tag v1-archived-2026-04-18 in the project repository for the record of what was claimed and why it was withdrawn.

This site exists because open research is supposed to be self-correcting. Finding a bug of this magnitude before external peer review — even at the cost of retracting one's own preprint — is what the process is for.
